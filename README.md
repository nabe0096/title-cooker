# バズタイトルクッカー

料理系YouTuber向けのタイトル自動生成ツール。

**公開URL**: https://tiny-otter-6659a0.netlify.app

---

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | ツール本体。これ1ファイルですべて完結 |
| `伸びるタイトル辞書.md` | タイトルパターンの元データ（現在は未使用） |

> タイトルパターンは `index.html` の中に直接埋め込んでいる。
> `伸びるタイトル辞書.md` は編集用メモとして残しているが、デプロイには影響しない。

---

## 使用ツール

| ツール | 役割 |
|---|---|
| **GitHub** (`nabe0096/title-cooker`) | ファイルの管理・保管 |
| **Netlify** (`tiny-otter-6659a0`) | Webサイトとして公開 |

---

## デプロイの流れ

```
index.html を編集
     ↓
GitHubにプッシュ（Claude が実行）
     ↓
Netlifyが自動検知
     ↓
自動デプロイ完了（数十秒）
```

基本的に社長が指示 → Claude が編集・プッシュ → 自動で公開される。

---

## 更新方法

Claude に「〇〇を変えて」と指示するだけでOK。
手動での操作は不要。

---

## Claudeへの指示ルール

- 社長が **「デプロイして」** と言ったら、その時点の `index.html` を GitHubにプッシュすること
- 編集のたびに自動プッシュはしない。**「デプロイして」の一言があって初めてプッシュする**
- 理由：Netlifyの無料プランは月300クレジット上限のため、無駄なデプロイを避ける

---

## タイトルパターンを追加したいとき

`index.html` 内の以下の4カテゴリにパターンを追記する：

- `king`：【王道・食欲直撃】型
- `gap`：【ギャップ・意外性】型
- `lazy`：【時短・ズボラ・コスパ】型
- `empathy`：【共感・解決】型

Claude に「〇〇型にこのパターンを追加して」と伝えるだけで対応可能。
